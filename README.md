# 💅 18 Karat Nails — Luxury Nail Art Studio

[![Live Site](https://img.shields.io/badge/Live-Site-green)](https://cheronoa.github.io/18karat-nails/)

## 🎯 Project Overview

A fully responsive, multi-page web application for **18 Karat Nails** — a luxury nail art studio based in Nairobi, Kenya. Built with HTML5, CSS3, JavaScript, and Bootstrap 5.

**Live Site:** [https://cheronoa.github.io/18karat-nails/](https://cheronoa.github.io/18karat-nails/)

## 📸 Features

- ✅ **8+ Pages** — Home, Services, Portfolio, About, Booking, FAQ, Testimonials, Contact
- ✅ **Luxury Pink Cheetah Theme** — Bold, elegant, and on-brand
- ✅ **Glass-morphism Effects** — Crisp, modern, and premium feel
- ✅ **JavaScript Interactivity** — Form validation + filterable gallery
- ✅ **Fully Responsive** — Works on desktop, tablet, and mobile
- ✅ **External CSS & JS** — Clean, organized, and maintainable
- ✅ **Scroll Animations** — Smooth fade-in effects as you scroll
- ✅ **SEO Optimized** — Meta tags, favicon, and accessibility features
- ✅ **Deployed on GitHub Pages** — Live on the internet

## 🛠️ Technologies Used

- HTML5
- CSS3 (with Glass-morphism)
- JavaScript (ES6+)
- Bootstrap 5
- Git & GitHub
- GitHub Pages (Deployment)
